# Personal Life Dashboard — Portfolio

## The Big Goal

Build a personal AI agent dashboard that manages different areas of my life (fitness, finance, goals, learning, projects) through a single natural language interface. Instead of context-switching between 5–10 apps, one prompt routes to the right domain and logs the activity.

## The Approach

Start with the smallest possible useful thing. Prove the core pattern works — frictionless logging, AI parsing, streak visualization — before scaling to more complex domains.

Each MVP is a learning, not just a product.

## Skills Demonstrated

- Product scoping and decision-making
- UX design for frictionless mobile experiences
- Full-stack development (Next.js, Supabase, Claude API)
- AI integration for natural language parsing
- Personal use cases as product research

## The Arc

| MVP | What it proves |
|---|---|
| Matcha Logger | Core pattern: log → visualize → feel rewarded |
| Fitness Logger | AI calorie parsing, photo input, more complex data model |
| Goal Tracker | Cross-domain reasoning, agent suggests actions |
| Personal Life Dashboard | Unified interface across all domains |
